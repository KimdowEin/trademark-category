# 商标分类工具

自动将商标局下载的 PDF 文件按规则分类归档。

## 安装

```bash
cargo build --release && cargo install --path
```

## 使用

```bash
# 基本用法（按注册号分类）
trademark-reply-category -i 下载文件夹 -o 归档文件夹

# 按回文类型分类
trademark-reply-category -i 下载文件夹 -o 归档文件夹 -m doc

# 自动解压 ZIP 文件后再处理
trademark-reply-category -x -i 下载文件夹 -o 归档文件夹
```

## 参数说明

| 参数 | 说明 |
|------|------|
| `-i` | 源文件夹（下载的文件所在位置） |
| `-o` | 输出文件夹（分类后的文件存放位置） |
| `-m` | 分类方式：`id` 按注册号，`doc` 按文档类型（默认 `id`） |
| `-x` | 开启自动解压（处理 ZIP 压缩包） |

## 文件命名规则

**输入文件**（来自商标局）：
```
注册证_12345678_001.pdf
受理通知书_87654321.pdf
```

**输出文件**：
```
12345678/注册证_12345678_001_[MD5值].pdf
```
- 按注册号分类：同一注册号的所有文件放入同一文件夹
- 按文档类型分类：同一类型的文件放入同一文件夹
